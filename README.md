# Groups

* `.` = any character
* `[xy]`, `(x|y)` = x or y
* `[^xy]` = not x or y


# Character Classes

* `/w` = `[a-zA-Z0-9]`
* `/d` = `[0-9]`
* `/l` = `[a-z]`
* `/u` = `[A-Z]`
* `/s` = `whitespace`
* `/W`, `/D`, `/L`, `/U`, `/S` = `[^/w]`, `[^/d]`, `[^/l]`, `[^/u]`, `[^/s]`


# Quantifiers

* `?` = zero or one
* `*` = zero or more
* `+` = one or more
* `{m,n}` = between m and n


# Lines
* `^` = line start
* `$` = line end 
* `\n`, `\t` = new line, new tab
